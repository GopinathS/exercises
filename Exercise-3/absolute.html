Absolute Positioning :

  • As we can see the below images for which the positioning is absolute , the elements are positioned exactly by specifying the top,left,right,bottom parameters .
    • The absolute positioning does not occupy any space.We can position the elements wherever we want using those parameters.After the positioning of any element, the other elements would be positioned differently with respect to the absolute positioned element.
  • As we can see below ,the images are overlapped which says the images are absolute positioned and does not occupy any space which made the second images to overlap.
Static Positioning :

  • The default positioning for all the elements is static position which occurs one by one as in document.

  • As we can see below ,the images are positioned as static and occurs where it normally would in the document.


RELATIVE Positioning :

  • As we can see the below images for which the positioning is relative , the elements are positioned exactly by specifying the top,left,right,bottom parameters .
    • The content of relatively positioned elements can be moved and overlap other elements, but the reserved space for the element is still preserved in the normal flow. The other elements would not be allowed to occupy that space which is the major difference with absolute positioning .
  • As we can see the below images are overlapped one on one which says the first image occupied some space which made the other image (absolute) to overlap on it.


Floating Left/Right :

A floated element will move as far to the left or right as it can. Usually this means all the way to the left or right of the containing element.
